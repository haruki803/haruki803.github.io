## 工夫点
・pcとスマホからストレスなく見れるレスポンシブルデザインにした点<br>
・アイテムが価格帯、服の種類、新着商品、おすすめ商品にカテゴライズされており、お客様が求めている商品を見つけやすくなっている点<br>
・スライドショーをページ上部に付けることで新着のコンテンツ、アイテムを満遍なく訪問者に告知できる点
